## Brainstorming Techniques

**Divergent Phase (never evaluate)**
- Classic brainstorm: quantity over quality
- Worst Possible Idea: generate terrible ideas → reverse them
- SCAMPER: Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse
- Random stimulus: add random word/image → force connections

**Convergent Phase (evaluate and select)**
- Dot voting on top ideas
- Impact/Effort matrix
- "Yes, and..." clustering
- Devil's Advocate challenge

## Session Flow

```
1. Warm-up (5 min) — lower inhibition
2. Problem framing — one clear How Might We question
3. Diverge (15 min) — generate without judgment
4. Cluster (5 min) — group similar ideas
5. Converge (10 min) — select most promising
6. Next step — assign owner and timeline
```

## YES AND Rule

During divergent phase:
- NEVER say "Yes, but..." (kills energy)
- ALWAYS say "Yes, AND..." (builds on ideas)
- No evaluation, no criticism, no "that won't work"

## SCAMPER Prompts

- Substitute: What if we replaced [element] with [alternative]?
- Combine: What if we merged [X] with [Y]?
- Adapt: How could [concept from another domain] apply here?
- Modify/Magnify: What if we made [element] 10x bigger/smaller/faster?
- Put to other uses: Who else could use this? When else?
- Eliminate: What if we removed [element] entirely?
- Reverse: What if we did the opposite?

## Rules

- Divergent phase: zero judgment, all ideas welcome
- Convergent phase: structured evaluation
- Energy is contagious — model enthusiasm
- Celebrate wild ideas explicitly
- No emojis in final outputs (Mantra IA-23)
- Never skip warm-up — psychological safety first
